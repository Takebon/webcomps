# Simple ball follow cursor

## Usage
```js
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">  
  <script src="https://cdn.jsdelivr.net/gh/Takebon/webcomps/tcogCursor/tcog-cursor.js"></script>
</head>
<body>
  <tcog-cursor
    size="80"
    fill-color="#d9ff00"
    stroke-color="#000"
    stroke-width="2"
    opacity=".1"
    drag=".15"></tcog-cursor>
</body>
</html>
```
